Como abrir esta demonstração localmente

Esta página é estática e não precisa instalar dependências.

cd /caminho/para/animalcheck
python3 -m http.server 8080
# abra http://localhost:8080/presentation/ no navegador

Se preferir, também é possível abrir o arquivo presentation/index.html diretamente no navegador. O servidor local é recomendado para simular melhor o comportamento de um site real.
